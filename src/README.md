# Selector Health Checker

Playwright-based tool to monitor selector health across public pages and detect broken selectors.

## Features (planned)
- Checks selectors across multiple jobs (URLs)
- Retries, timeouts, concurrency limit
- Structured logging
- JSON/CSV report output
- Exit code for CI usage

## Requirements
- Node.js (LTS recommended)
- npm
- Playwright browsers installed (via Playwright)
